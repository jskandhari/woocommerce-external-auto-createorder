=== WooCommerce External/Affiliate Products Auto Create Order ===
Contributors: Iskandariya Solutions
Tags: woocommerce
Stable tag: 1.0.0

== Description ==

This plugin allows you to automatically Add Order on External/Affiliate Product Button click, allowing you to track all the clicks and turn-overs.

== Installation ==

1. Upload the entire 'woocommerce-external-auto-createorder' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
